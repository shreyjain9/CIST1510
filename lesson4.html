Repeating Code with Loops

With loops, you can repeat a certain block of code as long as a condition remains to be true or is true.

for Loop

for (int i = 0; i < 7; i++) {
  System.out.println("i = " + i);
}

while Loop

int i = 0;
while (i < 7) {
  System.out.println(i);
  i++;
}

do-while Loop

int z = 0;
do {
  System.out.println(z);
  z++;
} while (z < 7);
