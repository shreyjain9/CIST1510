Repeating Code with Loops

Loops allow you to repeat a block of code as long as a condition is true.

for Loop

for (int i = 0; i < 5; i++) {
  System.out.println("i = " + i);
}

while Loop

int i = 0;
while (i < 5) {
  System.out.println(i);
  i++;
}

do-while Loop

int j = 0;
do {
  System.out.println(j);
  j++;
} while (j < 5);
