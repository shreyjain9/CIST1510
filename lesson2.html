Understanding Variables

Variables are containers or memory locations that store data. Variables in Java need to be declared with a specific data type.

Primitive Data Types

  • byte – 8-bit integer (-128 to 127)
  • short – 16-bit integer (-32,768 to 32,767)
  • int – 32-bit integer (e.g., 1, 100, -5)
  • long – 64-bit integer (e.g., large numbers like 1,000,000,000)
  • float – 32-bit decimal (e.g., 3.14f)
  • double – 64-bit decimal (e.g., 3.1415926535)
  • char – A single Unicode character (e.g., 'A')
  • boolean – T or F (True or False)

Example

int age = 17;
double gpa = 3.75;
char gr = 'A';
boolean isPass = true;

Reference Types

The most common reference type is String, which stores text:

String name = "Shrey";
