Understanding Variables

Variables are containers for storing data. Every variable in Java must be declared with a data type.

Primitive Data Types

  • int - for integers
  • double - for decimal numbers
  • char - for single characters
  • boolean - for true/false values

Example

int age = 20;
double gpa = 3.8;
char grade = 'A';
boolean isPassed = true;

Reference Types

The most common reference type is String, which stores text:

String name = "Alice";
