What Are Variables?

Variables are used to store data. Each variable has a type, which defines the kind of value it can hold.

Primitive Data Types

  • int - Integer values (e.g., 10, -3)
  • double - Decimal numbers (e.g., 3.14, -0.01)
  • char - A single character (e.g., 'A')
  • boolean - true or false

Examples

int age = 25;
double price = 9.99;
char grade = 'A';
boolean isJavaFun = true;

Reference Types

Strings and arrays are reference types. They store memory addresses of the actual data.

String name = "Alice";
int[] numbers = {1, 2, 3};
