What are Variables?

Variables are elements that store data in a Java program. They need a type.

Java Data Types

Data Type Example Description
int int a = 9; This manages whole numbers
double double p = 8.88; This manages decimal numbers
char char g = 'B'; This manages a single character
boolean boolean fun = false; This manages true or false values
String String n = "Sh"; This stores text
