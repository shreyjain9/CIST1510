What is Programming?

Programming is the process of writing instructions for a computer to follow.

Computers execute these instructions to perform tasks and solve problems.

Why Learn Java?

  • Java is widely used in web apps, Android apps, and enterprise software.
  • It runs on multiple platforms due to the Java Virtual Machine (JVM).
  • Java follows the Object-Oriented Programming (OOP) paradigm.

Writing Your First Java Program


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Explanation:

  • public class HelloWorld: Defines a class named HelloWorld.
  • public static void main(String[] args): Entry point of a Java program.
  • System.out.println("Hello, World!");: Prints text to the screen.

Activity

Modify the program to print your name instead of "Hello, World!"
