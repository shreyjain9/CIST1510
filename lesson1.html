What is Java?

Java is an extremely powerful programming language that's used to develop many applications, including web, desktop, and mobile applications. It is object-oriented, which allows Java to continue to be top choice for developers.

Why Java?

  • Write once, run anywhere
  • Widely used across industries and Versatile
  • A lot of support from the community
  • Amazing for beginners and Platform-Independent

Installing Java

  1. Download the JDK, or the Java Development Kit from Oracle.
  2. Install an IDE such as Eclipse.
  3. Finally, set up the environment variables.

First Java Program

  public static void main(String[] args) {
    System.out.println("Hello, my name is John!");
